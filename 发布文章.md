# CNKI助手：知网文献批量下载插件，全新升级

---

> 告别一篇一篇下载的痛苦，从油猴脚本到独立插件，更好用了。

---

## 为什么要做这个插件？

用过知网的同学都知道，下载论文是个体力活——一篇一篇点开、一篇一篇下载，遇到要下几十篇文献的时候，真的很崩溃。

之前我做了一个「知网硕博论文PDF下载」的油猴脚本，帮大家解决批量下载的问题，用的人还不少。但脚本毕竟有局限性：依赖油猴扩展、界面简陋、功能也不好扩展。

所以这次，我把它**重构成了一个独立的浏览器插件** —— **CNKI助手**。

---

## 有什么变化？

### 形态升级

从油猴脚本变成了独立的浏览器**侧边栏插件**，不再需要安装油猴，打开侧边栏就能用，和知网页面并排操作，体验好很多。

### 核心功能

| 功能 | 说明 |
|------|------|
| **批量收藏** | 一键收藏当前页全部文献，也可以单篇添加/取消 |
| **批量下载** | 批量获取 PDF 下载链接，一键批量下载 |
| **期刊等级** | 自动获取期刊等级标签（北大核心、CSSCI、SCI、EI 等），快速判断文献质量 |
| **摘要预览** | 点一下就能看摘要，不用再跳转详情页 |
| **复制信息** | 一键复制文献信息（标题、作者、期刊、日期） |
| **智能排序** | 支持按时间、被引量、下载量排序 |
| **WebVPN** | 支持 WebVPN 模式，校外访问也能用 |

### 浏览器兼容

支持所有 **Chromium 内核**浏览器：

- **Chrome** — 完全支持
- **Edge** — 完全支持
- **360极速浏览器** — 支持
- **QQ浏览器** — 支持
- **搜狗浏览器** — 支持

> 注：Firefox、Safari 暂不支持。

## 怎么安装？

目前还没上架应用商店，可以通过**开发者模式**加载安装：

1. **下载**插件压缩包，解压
2. **将解压后的文件夹放到一个固定位置**（比如「文档」目录下）
3. 打开浏览器，进入扩展管理页面
   - Chrome 用户：地址栏输入 `chrome://extensions/`
   - Edge 用户：地址栏输入 `edge://extensions/`
4. 开启右上角**「开发者模式」**
5. 点击**「加载已解压的扩展程序」**，选择插件文件夹
6. 在知网搜索页点击浏览器工具栏的 **CNKI助手** 图标即可使用

> **重要提示：** 安装后请不要删除或移动插件文件夹，否则插件会失效。

---

## 怎么更新？

下载新版本后，解压**覆盖**原来的文件夹，然后到扩展管理页面点击插件卡片上的**刷新按钮**即可，无需重新安装。
