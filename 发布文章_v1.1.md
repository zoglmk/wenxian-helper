# CNKI助手 v1.1 更新：引用格式一键复制，批量导出，写综述更顺手

---

> 这次主要为了让「整理参考文献」这件事少一点重复劳动。

---

## 这次更新了什么？

### 新增：引用格式一键复制

每篇文献卡片右侧的复制按钮，现在多了三种引用格式：

- **GB/T 7714**（国标，毕业论文最常用）
- **APA**（社科类常用）
- **MLA**（人文类常用）

直接复制粘贴到论文里就行，作者、卷期、页码、DOI 全都对得上知网官方格式——因为这次直接调用了知网的官方引用接口，不再本地拼装，准确度比之前高很多。

### 新增：批量导出

文献列表上方多了一个「导出」按钮，支持两类操作：

**一键复制多篇引用**

| 选项 | 说明 |
|------|------|
| 复制 GB7714 引用 | 把勾选的文献按 GB7714 格式拼好，全部复制到剪切板 |
| 复制 APA 引用 | 同上，APA 格式 |
| 复制 MLA 引用 | 同上，MLA 格式 |

适合**写综述时**把一批文献的参考文献一次性复制走。

**导出文献清单**

| 格式 | 用途 |
|------|------|
| **CSV** | 用 Excel 打开，方便筛选/统计 |
| **BibTeX** | 给 LaTeX / Zotero 导入 |
| **RIS** | 给 EndNote / NoteExpress 导入 |

包含标题、作者、期刊、卷期、页码、DOI、关键词、摘要等完整字段。

### 新增：验证码 / 额度满 智能暂停

之前批量下载遇到验证码或当日下载额度满时，插件会**连续尝试 5、6 次失败**才停下来——不仅浪费时间，还可能多消耗几次额度。

现在改成**第一时间识别并暂停**：

- 触发**验证码** → 自动暂停 + 新标签页打开验证码界面，输完验证码点「继续下载」从断点续传
- **额度已满** → 自动暂停 + 跳转到详情页查看
- 其他异常 → 暂停 + 自动打开页面让你看具体情况

不会再"白白失败一堆"了。

### 新增：使用说明 + 赞赏入口

- 空白页的使用说明加上了 v1.1 新功能简介
- 底部加了「☕ 赞赏」入口，如果觉得这个插件帮到你了，欢迎请作者喝杯咖啡 ☕

## 修复了什么？

| 问题 | 现状 |
|------|------|
| 触发验证码后会连续失败 5-6 次才停 | 第 1 次识别到就暂停 |
| 当日下载额度已满被显示成"验证码" | 准确识别为"额度已满" |
| 引用格式部分作者粘连、字段缺失 | 走知网官方接口，跟知网导出一致 |
| 多作者文献作者名连成一串（如"张三李四王五"） | 修复 |

---

## 怎么更新？

下载新版本后，解压**覆盖**原来的文件夹，然后到扩展管理页面点击插件卡片上的**刷新按钮**即可。

> 老用户更新后建议在文献列表页点一次「获取链接」，老数据会顺便修复作者粘连问题。

---

## 温馨提示

建议每次批量下载不超过 20 篇，下载完成后间隔一段时间再继续，避免触发知网的访问限制。
